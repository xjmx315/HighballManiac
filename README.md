# HighballManiac 

## Description

* 하이볼 레시피를 검색할 수 있는 사이트. 
* 내가 지금 가지고 있는 술을 입력하면 만들 수 있는 하이볼을 검색해줌. 
* 하이볼 도감을 만들어서 내가 만든 하이볼을 평가하고 수집할 수 있음. 
* 다른 사람들의 하이볼 후기를 확인할 수 있음. 
* 칵테일 바 같은 곳에서 광고, 레시피 공유 등을 할 수 있음. 
* 취향에 따른 하이볼 세트를 구성하여 컨셉에 맞는 하이볼을 추천해줌. 

## Base
* express.js
* dotenv
